\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_skeleton}{Skeleton}} class is used for skeleton creations and manipulations. 

\mbox{\hyperlink{class_skeleton}{Skeleton}} class is a class that keeps track of the image different frames where a specific skeleton may appears, its body key points, correspondent 3D space points, and other useful information skeleton-\/related.
