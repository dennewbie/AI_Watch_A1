\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_body_key_point}{Body\+Key\+Point}} class is used for body keypoint creations and manipulations. 

\mbox{\hyperlink{class_body_key_point}{Body\+Key\+Point}} class is a class that represents a Body Keypoint with its own x coordinate and y coordinate. This class inherits from \mbox{\hyperlink{class_point}{Point}} and adds the \char`\"{}confidence\char`\"{} detail in order to keep track of the Open\+Pose confidence on this particular point.
