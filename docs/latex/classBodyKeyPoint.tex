\doxysubsection{Detailed Description}
\mbox{\hyperlink{classBodyKeyPoint}{Body\+Key\+Point}} class is used for body keypoint creations and manipulations. 

\mbox{\hyperlink{classBodyKeyPoint}{Body\+Key\+Point}} class is a class that represents a Body Keypoint with its own x coordinate and y coordinate. This class inherits from \mbox{\hyperlink{classPoint}{Point}} and adds the \char`\"{}confidence\char`\"{} detail in order to keep track of the Open\+Pose confidence on this particular point.
