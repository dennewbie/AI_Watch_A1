The \mbox{\hyperlink{classCleanCommand}{Clean\+Command}} class is a class that implements a command to clean the build folder by deleting old files. 

The \mbox{\hyperlink{classCleanCommand}{Clean\+Command}} class is a class that implements a command to clean the build folder by deleting old files. It deletes all the images captured and computed when the current \char`\"{}\+JSONs\textquotesingle{} sending session with Kafka\char`\"{} is completed, as well as it deletes all the JSONs files produced in the meanwhile.
