\doxysubsection{Detailed Description}
\mbox{\hyperlink{classSystemCommand}{System\+Command}} class is a class that represents a command given by the program to the system. 

\mbox{\hyperlink{classSystemCommand}{System\+Command}} class is a class that represents a command given by the program to the system. Examples of commands are the following\+: start and run Open\+Pose, move files, and clean the build folder by deleting old files. \mbox{\hyperlink{classSystemCommand}{System\+Command}} implements a variant of the Command design pattern. Each concrete command is realized by a specific subclass that implements the parent abstract method.
